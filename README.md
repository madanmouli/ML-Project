# 🏠 House Price Prediction using Machine Learning
## 📌 Project Overview

This project predicts house prices using Machine Learning techniques. It uses the Random Forest Regressor algorithm for accurate house price prediction and ARIMA for future price forecasting. The application provides an easy-to-use interface where users can enter house details and get predicted prices instantly.
## ✨ Features

- 🏠 House Price Prediction
- 📈 Future Price Forecasting
- 🤖 Random Forest Regression Model
- 📊 Data Visualization
- 🌐 User Friendly Interface
- ⚡ Fast Predictions

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Statsmodels
- Flask
- HTML
- CSS

## 📂 Dataset

The dataset contains historical house prices with location, area, bedrooms, bathrooms, year built, and other important features used for prediction.
